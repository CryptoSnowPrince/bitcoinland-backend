OS: Ubuntu 22.04

## How to start

## How to run as development mode

```shell
yarn
yarn start
```

## How to run as test mode

```shell
yarn
yarn test
```

## How to run as product mode

```shell
yarn
yarn start-server
```

## How to stop server

```shell
yarn
yarn stop-server
```
